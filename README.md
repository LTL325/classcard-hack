## 📄 How to use
1. Compile index.ts
2. Enter `node index.js` and edit the generated config.json file
3. Enter `node index.js`
4. Invite bot to discord server.
5. Send `!setup` on discord server.
6. Just do as the bot tells you in the `#사용` channel.
